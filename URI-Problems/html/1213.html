Timelimit: 1

Dado qualquer inteiro n (1 ≤ n ≤ 10000) não divisível por 2 ou por 5, algum múltiplo de n deve ser um número que é uma sequência de números 1. Você deve então calcular e mostrar quantos dígitos tem o menor múltiplo de n que tem todos seus dígitos iguais a 1.

Entrada

A entrada consiste de vários casos de teste e termina com EOF. Cada caso de teste contém um inteiro n (1 ≤ n ≤ 10000) não divisível por 2 ou por 5.

Saída

Para cada caso de teste, imprima quantos dígitos tem o múltiplo de n que atende os requisitos acima.

3
7
